1. Asegúrate de tener instalado Python en tu computadora. Si no lo tienes instalado, puedes descargarlo desde la página oficial de Python.

2. Abre una terminal y navega hasta el directorio donde has descargado el código fuente de la aplicación.

3. Instala las librerías que se utilizan en la aplicación. Para ello, ejecuta el siguiente comando en la terminal:

```
pip install tkinter Pillow qrcode
```

Esto instalará las librerías necesarias para la aplicación, como `tkinter`, `Pillow` y `qrcode`.

4. Ejecuta la aplicación ejecutando el siguiente comando en la terminal:

```
python main.py
```

Esto iniciará la aplicación y mostrará una ventana con un campo de texto, un botón y una imagen del código QR.

5. Para generar un nuevo código QR, simplemente ingrese la información deseada en el campo de texto y haga clic en el botón "Generar". El código QR generado se mostrará automáticamente en la ventana.

6. Cuando hayas terminado de usar la aplicación, puedes cerrarla haciendo clic en el botón "Cerrar" en la ventana o simplemente cerrando la ventana.
